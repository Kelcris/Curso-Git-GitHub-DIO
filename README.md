# Anotações sobre o curso Introdução ao Git/GitHub 

**Comandos básicos para um bom desempenho no terminal:**

*Windows:*

>cd (navegar entre pastas)

>cd.. (retorna um nível na navegação)

>cls (limpa a tela)

>dir (listar)

>mkdir (criar)

>del (deletar arquivos)

>rmdir (deleta repositórios)

*Unix:*

>cd (navegar entre pastas)

>cd.. (retorna um nível na navegação)

>clear (limpa a tela)

>ls (listar)

>mkdir (criar)

>rm-rf (deletar)
